Settings

Providers

We use PROVIDER_URL as primary API, then we fallback to the other providers in order.

Download Quality

Override the default audio quality configured via QUALITY environment variable.

Theme Selector

Override the default application theme configured via environment variable.

Genre Mapping

Customize how MusicBrainz genres are normalized. Maps sub-genres to main genres.

Mood List

Default moods available for selection. Stored as JSON array.

Style List

Default styles available for selection. Stored as JSON array.

Genre Separator

Separator used to split multiple genres when writing to audio file tags. Requires restart to take effect.

Force Downloads

Re-download tracks even if they already exist on disk. Deletes existing files before re-downloading.
